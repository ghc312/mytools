# 🛒 网购防坑小管家

618 买了一堆，退了吗？价保过期了吗？维权话术写对了吗？

**纯前端 HTML 看板，双击打开，零安装。**

## 功能

- **订单录入** — 选择平台、填写商品和购买日期，自动计算退货截止日
- **退货倒计时** — 红/橙/绿三色预警，今到期标红、3 天内标橙、安全标绿
- **维权模板** — 按商品类别（电子/服装/食品/家居/美妆）一键生成专业投诉文案
- **数据看板** — 进行中、今到期、窗口内、已过期四维统计
- **本地存储** — 数据存浏览器 localStorage，离线可用、秒开
- **导入/导出** — JSON 备份，换设备不丢

## 使用方式

1. 双击 `index.html` 用浏览器打开
2. 点「添加订单」，填商品名、金额、购买日期
3. 退货到期前系统红色预警，点击「维权模板」可直接复制文案发给客服

## 维权模板亮点

- 自动填充商品名、金额、日期
- 引用《消费者权益保护法》《食品安全法》具体条款
- 覆盖电子/服装/食品/家居/美妆五大品类
- 附加 12315 投诉威慑
